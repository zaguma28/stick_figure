# SPEC.md

# MVP 実装仕様書（Single Source of Truth）

## 0. 安全ルール（エージェント運用）

- rm -rf / del / format / diskpart 等の破壊的操作は禁止
- 削除が必要な場合は「コマンド提示→承認待ち」
- プロジェクトはgitで管理し、マイルストーンごとにコミット
- 各マイルストーンでビルド/実行テストを行い、結果をログ化する

## 1. 画面・操作（縦持ち）

UI（画面右側に集約）

- 移動：左スティック
- 攻撃：右ボタン（軽攻撃のみ）
- ロール（回避）：右ボタン
- ガード：右ボタン（押しっぱ）
- パリィ：右ボタン（単押し）
- スキル：2ボタン（CT制）
- 回復（エスト）：1ボタン（回数制）

ターゲット補助

- 近い敵へ弱い吸着（ロックオンはMVPでは任意）

## 2. プレイヤー初期ステータス

- HP：100
- スタミナ：100
- スタミナ回復：22/秒
- スタミナ回復ディレイ：行動後0.25秒
- 被弾後無敵：0.35秒
- 移動速度：4.6

### 2.1 スタミナ消費

- 通常攻撃：18 / 18 / 26（3段）
- ロール：28（無敵0.24秒、硬直0.14秒、距離2.8）
- パリィ：24
- エスト回復：16

## 3. ガード（弱/強）

### 3.1 弱ガード（全員標準）

- ダメ軽減：30%
- ガード被弾スタミナ消費：22 / 1ヒット
- スタミナ0でガードブレイク：硬直0.6秒

### 3.2 強ガード（ビルドで解放）

- ダメ軽減：65%
- ガード被弾スタミナ消費：-35%（パッシブでさらに軽減可能）

## 4. パリィ

- 入力受付：0.18秒（短い）
- 成功：敵体幹に大ダメ（雑魚 -45 / ボス -30）
- 失敗：硬直0.35秒

## 5. 体幹（Poise）

- 雑魚：Poise 40（回復8/秒、非被弾中）
- 盾兵：Poise 70
- ボス：Poise 220
- 体幹0でダウン：雑魚1.2秒 / ボス1.5秒

## 6. 通常攻撃・スキル（MVP）

### 6.1 通常攻撃（3連）

- ダメ：10 / 10 / 18
- 体幹：8 / 8 / 14
- 3段目硬直：0.30秒（振った責任）

### 6.2 スキル

- 直線突き：ダメ42、CT6.0s、体幹18

## 7. 敵（6種）基本値とスケーリング

スケーリング

- HP係数：S = 1 + 0.08\*(floor-1)
- ダメ係数：D = 1 + 0.06\*(floor-1)

敵（ベース）

- Charger：HP55、接触22、突進予兆0.35s
- Shooter：HP40、弾16、間隔1.3s
- Spreader：HP48、弾14×3、間隔2.2s、±18°
- Shield：HP95、正面軽減80%、近接20
- Bomber：HP35、爆発32、予兆0.45s
- Summoner：HP70、弾12、雑魚召喚（HP25/ダメ12）×2、CT6.5s

## 9. 報酬（ローグの核）

### 9.1 タグ

dodge / guard / poise / bullet / crit / parry / bleed

### 9.2 報酬生成（3択）

重み付け

- 所持タグが2個以上：該当系統出現×1.8
- 所持タグが1個：×1.3
- ただし3択のうち1枠は別系統を混ぜる（乗り換え余地）

### 9.3 詰み防止（9F天井）

9Fまでに以下のどれかを必ず候補へ混ぜる（未所持なら強制）

- 強ガード解放
- 弾消し手段
- ロール安定化（無敵+ or 消費-）
- 低HP火力（狂戦士系）

### 9.4 戦い方が変わる“解放報酬”（例）

- 強ガード解放：ガード軽減 30%→65%
- 弾消しスラッシュ：CT14s、扇状120°、半径3.2
- 致命の印：体幹0の敵へ次の一撃×2.2（代償：通常攻撃-10%）
- 回避無敵+0.05s（レア）
- ロール消費-15%（回避ビルドの土台）
